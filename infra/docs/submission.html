SlotPilot

Tagline: AI-assisted Solana transaction control tower for Jito bundle submission, Yellowstone lifecycle proof, failure classification, and retry decisions.

What It Does

SlotPilot treats Solana transaction delivery as an operational lifecycle instead of a single RPC call. It builds signed transfers, prices Jito tips, submits through Jito bundles, watches live Yellowstone data, classifies failures, and records append-only JSONL evidence for every attempt.

Technical Highlights

  • TypeScript/Node modular monolith with no database.
  • Dockerized Linux runtime for live Yellowstone compatibility.
  • Live preflight passes Solana RPC, Jito tip accounts, Jito leader planning, OpenAI mode, and Solinfra Yellowstone slot streaming.
  • OpenAI retry agent returns strict JSON decisions and never receives private keys.
  • Jito bundle integration returns real bundle IDs and records signatures, slots, tips, and decision attribution.
  • Strict live verifier rejects mock or incomplete evidence packages.
